Завдання 2

Написати програму сортування списку у порядку зростання методом бульбашки.

# Функція сортування методом бульбашки
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False  # Оптимізація: прапорець для перевірки, чи були зміни
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                # Міняємо елементи місцями
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        # Якщо жодного обміну не було, список уже відсортований
        if not swapped:
            break

# Крок 1: Інтерактивне введення елементів списку
arr = list(map(int, input("Введіть елементи масиву через пробіл: ").split()))

# Викликаємо функцію сортування
bubble_sort(arr)

# Виводимо відсортований список
print("Відсортований список:", arr)

Завдання 3

Уявіть список - не дуже довгий, не дуже складний, просто звичайний список, що містить деякі цілі числа. Деякі з цих чисел можуть повторюватись, і це ключ до розгадки. Ми не хочемо повторень. Ми хочемо, щоб їх видалили.

Ваше завдання – написати програму, яка видаляє всі дублікати чисел зі списку. Ціль полягає в тому, щоб скласти список, в якому всі числа зустрічаються не більше одного разу.

Примітка

припустимо, що вихідний список жорстко закодований всередині коду - Вам не потрібно вводити його з клавіатури. Звичайно, Ви можете покращити код і додати частину, яка може вести діалог з користувачем і отримувати від неї всі дані..

Підказка

ми рекомендуємо Вам створити новий список як тимчасову робочу область - вам не потрібно оновлювати список in situ.

Ми не надали тестових даних, оскільки було б надто просто. Натомість ви можете використовувати наш скелет.

my_list = [1, 2, 4, 4, 1, 4, 2, 6, 2, 9]

# Створюємо новий список, додаючи до нього тільки унікальні елементи
unique_list = []
for item in my_list:
    if item not in unique_list:
        unique_list.append(item)

# Виведення списку з унікальними елементами
print("The list with unique elements only:")
print(unique_list)
The list with unique elements only:
[1, 2, 4, 6, 9]

Пояснення: У списку my_list містяться повторювані числа. Ми створюємо новий список unique_list, куди додаємо лише ті елементи, які ще не зустрічалися (перевіряємо за допомогою операції if item not in unique_list). Після проходження по всьому вихідному списку, виводимо новий список без дублікатів.

Напишіть код, використоувуючи генератори списків, який створює матрицю 8х8 з пустими клітинками (пуста клітинка задається як “_“) для задання шахівниці і розставте чотири тури по кутках шахівниці

def create_chessboard():
  """Створює шахівницю 8x8 з турами по кутах.

  Returns:
    list: Двовимірний список, що представляє шахівницю.
  """

  # Створюємо пусту шахівницю за допомогою генератора списків
  chessboard = [['_' for _ in range(8)] for _ in range(8)]

  # Розставляємо тури по кутах
  chessboard[0][0] = '♖'
  chessboard[0][7] = '♖'
  chessboard[7][0] = '♖'
  chessboard[7][7] = '♖'

  return chessboard

def print_chessboard(chessboard):
  """Виводить шахівницю в консоль."""
  for row in chessboard:
    print(' '.join(row))

# Основна програма
if __name__ == "__main__":
  board = create_chessboard()
  print_chessboard(board)

list_1 = ["A", "B", "C"]
list_2 = list_1
list_3 = list_2

del list_1[0]
del list_2[0]

print(list_3)
['C']

Усі три змінні (list_1, list_2, list_3) посилаються на один і той самий список. Видаляючи елементи через одну змінну, ви змінюєте список для всіх змінних. В результаті, коли виводиться list_3, він містить тільки залишковий елемент [‘C’].

list_1 = ["A", "B", "C"]
list_2 = list_1
list_3 = list_2

del list_1[0]
del list_2

print (list_3)
['B', 'C']

У цьому коді створюються три списки:

  1. list_1 - це список, що містить елементи [“A”, “B”, “C”].
  2. list_2 - це змінна, яка посилається на той самий об’єкт, що й list_1. Тобто list_2 тепер також є [“A”, “B”, “C”].
  3. list_3 - це ще одна змінна, яка також посилається на той самий об’єкт, що й list_2 і list_1. Отже, list_3 також є [“A”, “B”, “C”].

Далі виконується два видалення:

  1. del list_1[0] - видаляє перший елемент зі списку list_1, який тепер виглядає як [“B”, “C”]. Оскільки list_2 і list_3 також посилаються на той самий список, вони теж будуть відображати [“B”, “C”].

  2. del list_2 - видаляє змінну list_2, але сам об’єкт списку залишається в пам’яті, і на нього все ще посилаються list_1 та list_3.

Коли виконується print(list_3), буде виведено [“B”, “C”], оскільки list_3 все ще посилається на той самий список, на який вказують list_1 і list_2 до його видалення.

list_1 = ["A", "B", "C"]
list_2 = list_1
list_3 = list_2

del list_1[0]
del list_2[:]

print(list_3)
[]

list_1 = [“A”, “B”, “C”]: створюється список, який містить три елементи: “A”, “B” і “C”. list_2 = list_1 та list_3 = list_2: створюються нові змінні, але вони не містять копій списку list_1. Натомість вони є просто посиланнями на той самий список. Це означає, що всі три змінні вказують на один і той же об’єкт в пам’яті. Видалення елементів:

del list_1[0]: видаляється перший елемент (з індексом 0) зі списку list_1. Оскільки list_2 та list_3 посилаються на той самий список, видалення відбувається і в них. del list_2[:]: видаляє всі елементи зі списку list_2. Поскольку list_2 и list_3 ссылаются на один и тот же список, удаление произойдет и в list_3. Виведення результату:

print(list_3): виводить на екран вміст списку list_3, який після всіх операцій буде пустим списком [].

Чому виводиться пустий список? Оскільки всі три змінні (list_1, list_2, list_3) посилаються на один і той же список, будь-які зміни, внесені в один з цих списків, автоматично відображаються в інших. Коли ми видаляємо всі елементи зі списку list_2 (а отже, і зі списків list_1 та list_3), то в результаті всі три списки стають порожніми.

list_1 = ["A", "B", "C"]
list_2 = list_1[:]
list_3 = list_2[:]

del list_1[0]
del list_2[0]

print(list_3)
['A', 'B', 'C']

У цьому коді створюються три списки, але на цей раз вони копіюються, а не посилаються на один і той самий об’єкт:

  1. list_1 - це список, що містить елементи [“A”, “B”, “C”].
  2. list_2 = list_1[:] - створює новий список list_2, який є копією list_1. Тепер list_2 містить [“A”, “B”, “C”], але це окремий об’єкт у пам’яті, який не пов’язаний з list_1.
  3. list_3 = list_2[:] - створює ще один новий список list_3, який є копією list_2. Тепер list_3 також містить [“A”, “B”, “C”].

Далі виконуються два видалення:

  1. del list_1[0] - видаляє перший елемент з list_1, який тепер виглядає як [“B”, “C”]. list_2 та list_3 не змінюються, оскільки вони є окремими копіями.

  2. del list_2[0] - видаляє перший елемент з list_2, який тепер виглядає як [“B”, “C”]. Однак list_1 і list_3 все ще залишаються незмінними. list_1 - [“B”, “C”], а list_3 - [“A”, “B”, “C”].

Коли виконується print(list_3), буде виведено [“A”, “B”, “C”], оскільки list_3 не зазнав змін і залишився копією початкового списку list_1.
