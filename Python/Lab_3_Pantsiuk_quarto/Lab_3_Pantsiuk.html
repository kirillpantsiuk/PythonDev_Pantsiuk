Завдання № 1

Написати програму, яка обчислює значення функції Гауса: \[f(x) = \frac{1}{\sigma \sqrt{2 \pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}\]

import math

# Визначаємо функцію Гаусса
def gauss_function(x, mean, std_dev):
    return (1 / (std_dev * math.sqrt(2 * math.pi))) * math.exp(-0.5 * ((x - mean) / std_dev) ** 2)

# Приклад використання
x = 1.0    # значення змінної x
mean = 0.0   # математичне сподівання (середнє значення)
std_dev = 1.0  # стандартне відхилення

# Обчислення значення функції Гаусса
result = gauss_function(x, mean, std_dev)
print(f"Значення функції Гаусса для x={x}, mean={mean}, std_dev={std_dev}: {result}")

# Введіть значення для змінних a і b

print("2. Віднімання")

print("4. Ділення")

# Зчитування вибору користувача
operation = input("Введіть номер операції (1/2/3/4): ")

# Виконання обраної операції та виведення результату

    # Виведення результату додавання

Ваше завдання – завершити код, щоб оцінити такий вираз: \[\frac{1}{x + \frac{1}{x + \frac{1}{x + \frac{1}{x + \frac{1}{x}}}}}\]

Результат має бути присвоєний y.

Ви можете використовувати додаткові змінні для зменшення виразу (але це не обов’язково). Ретельно тестуйте свій код.

Тестові дані

# Введіть значення для x
x = float(input("Enter value for x: "))

# Обчислення виразу
# Розкладаємо вираз на декілька кроків для зручності
# Починаємо з самого внутрішнього виразу
inner4 = x + 1 / x
inner3 = x + 1 / inner4
inner2 = x + 1 / inner3
inner1 = x + 1 / inner2
y = 1 / inner1

# Виведення результату
print("y =", y)

Пояснення:

  1. inner4: Обчислює найглибший вкладений вираз у дробі.
  2. inner3: Використовує результат inner4 для обчислення наступного рівня.
  3. inner2: Використовує результат inner3 для обчислення наступного рівня.
  4. inner1: Використовує результат inner2 для обчислення наступного рівня.
  5. y: Обчислює остаточний результат виразу, використовуючи inner1.

Ваше завдання - підготувати простий код, здатний порахувати час закінчення періоду часу, вираженого в кількості хвилин (воно може бути як завгодно великим). Час початку вказується у вигляді пари годин (0..23) та хвилин (0..59). Результат повинен бути надрукований на консолі.

Наприклад, якщо подія починається о 12:17 і триває 59 хвилин, вона закінчується о 13:16.

Не турбуйтеся про будь-які недоліки у вашому коді –- це нормально, якщо він приймає неприпустимий час -– найважливіше, що код видає дійсні результати для правильних вхідних даних.

Ретельно перевірте код. Підказка: використання оператора % може бути ключем до успіху.

# Введіть час початку події у правильному форматі
hour = int(input("Введіть час початку (години, 0-23): "))

# Виведення результату
print(f"Подія закінчиться о: {end_hour:02}:{end_minute:02}")

Введення числа:

  • input("Enter a number: "): Ця частина коду відображає на екрані повідомлення “Введіть число”. Це підказка для користувача, яке число він має ввести.
  • Користувач вводить 2 і натискає Enter: Коли користувач бачить це повідомлення, він вводить число 2 і натискає клавішу Enter. Введений текст “2” зберігається в пам’яті комп’ютера.

Перетворення в ціле число:

  • int(): Ця функція бере введений текст (рядок) і перетворює його на ціле число. Тобто, рядок “2” перетворюється на число 2.
  • Присвоєння змінній: Отримане число 2 присвоюється змінній x. Тепер змінна x містить числове значення 2.

Множення рядка:

  • x * "5": Ця частина коду трохи незвичайна. Вона означає, що рядок “5” повторюється стільки разів, яке число зберігається в змінній x.
  • Повторення рядка: Оскільки x дорівнює 2, рядок “5” повторюється два рази. Тобто, ми отримуємо новий рядок “55”.

Виведення результату:

  • print(): Ця функція виводить на екран результат попередніх операцій. В нашому випадку, вона виводить рядок “55”, який ми отримали в результаті повторення рядка “5”.
x = int(input("Enter a number: "))  # Пользователь вводит 2
print(x * "5")

x = input("Enter a number: ")  # the user enters 2
